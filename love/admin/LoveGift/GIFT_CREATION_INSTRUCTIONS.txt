Follow these steps to create a personalized, loving chatbot gift for your special someone:

═══════════════════════════════════════════════════════════════════════════════
📋 STEP 1: CUSTOMIZE THE GIFT
═══════════════════════════════════════════════════════════════════════════════

1. Run the customization script:
   powershell -ExecutionPolicy Bypass -File "customize_gift.ps1"

2. Answer the questions:
   • Enter your loved one's name
   • Choose a custom greeting message
   • Decide if music should play
   • Choose if photos should be available
   • Set conversation length
   • Pick a theme (Hearts, Valentine's, Anniversary)

3. The script will create:
   • love_gift_chatbot_custom.ps1 (personalized chatbot)
   • README_LoveGift_Personal.txt (custom instructions)

═══════════════════════════════════════════════════════════════════════════════
📦 STEP 2: COMPILE TO EXECUTABLE (OPTIONAL)
═══════════════════════════════════════════════════════════════════════════════

For a professional, standalone gift:

1. Install PS2EXE module (run as Administrator):
   Install-Module -Name PS2EXE

2. Run the compilation script:
   powershell -ExecutionPolicy Bypass -File "compile_to_exe.ps1"

3. This creates: LoveGift_Chatbot.exe (standalone executable)

═══════════════════════════════════════════════════════════════════════════════
🎁 STEP 3: PACKAGE THE GIFT
═══════════════════════════════════════════════════════════════════════════════

Create a beautiful gift package with:

REQUIRED FILES:
• LoveGift_Chatbot.exe (or love_gift_chatbot_custom.ps1)
• README_LoveGift_Personal.txt
• Run_Love_Chatbot.bat (for .ps1 version)

OPTIONAL ENHANCEMENTS:
• heart_image.txt (for photo feature)
• A handwritten love note
• Flowers or chocolates
• USB drive or folder with romantic music

═══════════════════════════════════════════════════════════════════════════════
💝 STEP 4: PRESENT THE GIFT
═══════════════════════════════════════════════════════════════════════════════

1. Package everything in a nice folder or USB drive
2. Include the personalized README
3. Add a personal note explaining the gift
4. Present it with love and affection!

🚨 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

If the chatbot doesn't work:
• Make sure PowerShell execution is allowed
• Check Windows version compatibility
• Ensure .NET Framework is installed
• Try running as Administrator

For compilation issues:
• Install PS2EXE module correctly
• Check PowerShell version
• Ensure all paths are correct
